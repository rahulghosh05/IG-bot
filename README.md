## What It Does

This Python bot uses **Selenium** to:
- Log into my Instagram account
- Visit @yocaloffgod’s profile
- Open recent posts
- Leave one of 25+ rotating, meaningful comments from a custom bank
- Do it all like a human: timed, random, varied
